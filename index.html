My Journey to the Field of Technology

It seemed like the transition into becoming a "programmer" would be an easy task. I studied math in college and believed that this would be a good start to my career in the tech field. But to be a software developer, I should at least learn how to code. So, I learned how to code. I learned Python and noticed that I was sort of right, knowing math did help me with programming. Boolean logic and algorithms, all things that looked very familiar to me from my college days. However, I have struggled with what projects should I do. What can I make that has not been done already? This thought has been lingering over my head, a barrier preventing me from progressing further. Therefore, I have decided to focus on working on a project daily, regardless if it has been done or not. The goal is to build a portfolio. As cheesy as this may sound, thanks for joining me as complete my journey into the field of technology.

Project 2 – Responsive Web Design

I will continue to work on this website. However, I will now use bootstrap to have the website work on different devices. Many of these things I learned from reading online. W3 schools and the Bootstrap websites were my main resources in creating the menu bar.

As I incorporated bootstrap into the site, I started to add more features to the websites. However, I can came into some issues and learned that jquery is used to make the dropdown on the navbar work properly.

  1. Bootstrap Skills
  2. Learning the Grid
    • The page is "split" into 12 columns
    • Using col-lg, col-sm, col-md, col-xs
    • Using @media when we want different layouts for different media devices
  3. Creating a Menu
    • Using the navbar tag
    • Bootstrap recognizes the navbar and navbar-default class
    • Use dropdown-menu class for a drop down menu
    • dropdown-menu class works with navbabar-default at least when using
      bootstrap.min.css
    • Jquery was needed to make the dropdown menu work.
